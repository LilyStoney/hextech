# RiotApi

Ruby wrapper for Riot Api endpoints.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'riot_api'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install riot_api

## Usage

You'll need the environment variable:

```
RIOT_API_KEY='code-goes-here'
```
